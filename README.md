# Ridhan Parvendhan Portfolio

Evidence-first portfolio for data analytics, machine learning, AI-agent orchestration, and workflow automation projects.

Live site: https://ridhan-portfolio.vercel.app/

## Featured Evidence

- **AI Ops Workflow Automation Platform:** bounded LangGraph agent, RAG, explicit tools, evaluation, tracing, and approval controls
- **DirectDebit IQ:** auditable fintech risk scoring and governed retry scheduling
- **PayGuard:** explainable fraud-detection prototype with tests, CI, Docker, and SHAP
- **Baltic Commerce Intelligence:** reproducible Python, SQL, dbt, SQLite, Excel, and statistics case study

Project cards link to the corresponding repositories and live demos. Claims are intentionally scoped to implemented, reviewable evidence.

## Run Locally

```bash
npm ci
npm run dev
```

## Verify

```bash
npm run build
```
